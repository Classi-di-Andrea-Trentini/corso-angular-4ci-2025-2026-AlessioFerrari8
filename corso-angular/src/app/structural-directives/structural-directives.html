Direttive strutturali

Le direttive strutturali ci consentono di modificare in modo dinamico il DOM (struttura) della pagina, ovvero aggiungere o rimuovere tag in modo dinamico

La direttiva if

if ci consente di visualizzare o meno oppure scegliere fra due blocchi HTML al verificarsi di una condizione.

Esempio 1

Il paragrafo in basso viene aggiunto al DOM se il valore del signal è true, altrimenti non viene aggiunto nulla.

@if (visualizza()) {

Questo è un paragrafo che appare e scompare

}

Esempio 2

Con questo esempio decidiamo se visualizzare un'immagine oppure una scritta in base alla scelta fatta dall'utente ad esempio con button.

@if (immagineOTesto() == 'immagine') { } @else {

Questo testo è visualizzato in alternativa all'immagine

}
